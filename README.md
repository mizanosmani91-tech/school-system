## 🚀 ইনস্টলেশন

### প্রয়োজনীয়তা
- PHP 7.4+ (PHP 8.x সুপারিশকৃত)
- MySQL 5.7+ / MariaDB 10.3+
- Apache/Nginx
- cURL, PDO, mbstring, GD extensions

### ধাপসমূহ

**১. ফাইল আপলোড করুন**
```
htdocs/ বা www/ ফোল্ডারে school_system/ ফোল্ডারটি আপলোড করুন
```

**২. ইনস্টলেশন উইজার্ড চালু করুন**
```
http://yourdomain.com/school_system/install/
```

**৩. ডাটাবেস তথ্য দিন এবং Install ক্লিক করুন**

**৪. লগইন করুন**
```
URL: http://yourdomain.com/school_system/login.php
Username: admin
Password: password
```

**৫. সেটিংসে যান এবং আপডেট করুন**
- প্রতিষ্ঠানের নাম, ঠিকানা
- AI API Key (Anthropic Claude)

## 🤖 AI সহকারী সেটআপ

১. https://console.anthropic.com এ যান
২. API Key তৈরি করুন (sk-ant-...)
³. Settings → AI ও SMS সেটিংস → API Key বসান

**AI যা করতে পারে:**
- পাঠ পরিকল্পনা তৈরি
- পরীক্ষার প্রশ্নপত্র
- SMS বার্তা রচনা
- অভিভাবক চিঠি লেখা
- ইসলামী বিষয় ব্যাখ্যা

## 📁 ফাইল স্ট্রাকচার

```
school_system/
├── includes/
│   ├── config.php       ← ডাটাবেস কনফিগ
│   ├── functions.php    ← কোর ফাংশন
│   ├── header.php       ← হেডার টেমপ্লেট
│   └── footer.php       ← ফুটার টেমপ্লেট
├── modules/
│   ├── student/         ← ছাত্র মডিউল
│   ├── teacher/         ← শিক্ষক মডিউল
│   ├── attendance/      ← উপস্থিতি মডিউল
│   ├── exam/            ← পরীক্ষা মডিউল
│   ├── fees/            ← ফি মডিউল
│   ├── notice/          ← নোটিশ মডিউল
│   ├── parent/          ← অভিভাবক পোর্টাল
│   └── ai/              ← AI সহকারী
├── api/                 ← AJAX API endpoint
├── assets/              ← CSS, JS, Images
├── install/             ← ইনস্টলেশন উইজার্ড
├── index.php            ← ড্যাশবোর্ড
├── login.php            ← লগইন
├── logout.php           ← লগআউট
└── settings.php         ← সেটিংস
```
